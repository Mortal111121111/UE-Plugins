多表合表插件

一些常用的小功能 UI

- TabBar ： 目前使用侵入式解决，可以利用 Object 外置逻辑做到完全无感，也不需要使用自己提高的类，但是可能毕竟费
- SecondConfirm
- AutoWriterRichText ： 聪爷的插件
- KTimerText : 聪爷的插件
- SkipNumber : 数值动态变化累加控件，并使用 C++ 动态实现动画
